# WOE-070: Telemetry HUD -- Scaffold, Toggle, Reticle

**ID:** WOE-070
**Phase:** UI -- Telemetry HUD
**Depends on:** WOE-063
**UI Spec:** SS5.2

---

## Description

Create the Telemetry HUD `CustomPaint` overlay layer (Layer 3 in the AR Stack per SS4.1). This ticket sets up the scaffold: a full-screen `CustomPaint` widget, a Riverpod provider to toggle HUD visibility (default: ON), and the simplest HUD element -- the center reticle crosshair.

Subsequent tickets (WOE-071 through WOE-076) add individual HUD elements to this scaffold.

---

## Definition of Done

- [ ] `TelemetryHud` widget in `lib/shared/telemetry_hud.dart`
- [ ] Full-screen `CustomPaint` that paints HUD elements
- [ ] `hudVisibilityProvider` Riverpod provider (default: `true`); persisted in SharedPreferences key `hud_visible`
- [ ] When visibility is OFF, `TelemetryHud` renders nothing (returns `SizedBox.shrink()`)
- [ ] Reticle: boresight crosshair at screen center -- four 8dp arms with 12dp gap
- [ ] Reticle color: `hudPrimary` at 50% opacity
- [ ] Integrates into AR screen as Layer 3 (above WebView, below UI chrome)
- [ ] Widget test: reticle paints at center when visibility ON; nothing when OFF

---

## Sub-tasks

- [ ] Create `lib/shared/telemetry_hud.dart`
- [ ] Define `HudPainter extends CustomPainter` with initial `paintReticle()` method
- [ ] `paintReticle(Canvas, Size)`: draw 4 lines from center outward (up/down/left/right), each 8dp long, starting 6dp from center (12dp gap = 6dp on each side)
- [ ] Use `Paint()..color = tokens.hudPrimary.withOpacity(0.5)..strokeWidth = 1.5..style = PaintingStyle.stroke`
- [ ] `TelemetryHud` widget: `Consumer` that watches `hudVisibilityProvider`; returns `CustomPaint(painter: HudPainter(...), size: Size.infinite)` or `SizedBox.shrink()`
- [ ] Create `lib/shared/hud_visibility_provider.dart`: `@riverpod` notifier with `toggle()` and `bool` state; reads/writes `hud_visible` in SharedPreferences
- [ ] Run `dart run build_runner build`
- [ ] Add `TelemetryHud` to AR screen `Stack` between globe layer and UI chrome layer
- [ ] Widget test: mock provider to ON, verify `CustomPaint` renders; mock to OFF, verify `SizedBox.shrink`
