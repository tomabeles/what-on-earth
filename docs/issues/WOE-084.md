# WOE-084: Calibration Screen UI

**ID:** WOE-084
**Phase:** UI -- Onboarding
**Depends on:** WOE-063, WOE-020
**UI Spec:** SS4.8

---

## Description

Implement the standalone Calibration screen UI. This screen is used by both Onboarding step 3 (WOE-046) and Settings -> Sensor (WOE-083). Full-screen dark surface with an animated figure-8 device silhouette, a confidence ring, and Done/Restart buttons.

The calibration algorithm is in WOE-047. This ticket builds only the UI shell, accepting confidence as a stream/provider input. Can be tested with a mock confidence value.

---

## Definition of Done

- [ ] `CalibrationScreen` in `lib/onboarding/calibration_screen.dart`
- [ ] Full-screen, `surfacePrimary` background (dark)
- [ ] Animated device silhouette performing figure-8 motion, 4-second loop cycle
- [ ] Instruction text: "Move your device in a slow figure-8 pattern."
- [ ] Confidence ring: circular progress indicator (0-100%)
- [ ] Ring color: `hudDanger` 0-39%, `hudWarning` 40-79%, `hudPrimary` 80-100%
- [ ] Confidence label: percentage displayed inside the ring
- [ ] "Done" button: enabled when confidence >= 80%, disabled (greyed) below
- [ ] "Restart" text button: calls a restart callback (resets to 0%)
- [ ] Accepts `Stream<double>` or provider for confidence (0.0-1.0)
- [ ] Widget test: Done button disabled at 50%, enabled at 85%; ring color correct at each threshold

---

## Sub-tasks

- [ ] Create `lib/onboarding/calibration_screen.dart`
- [ ] Figure-8 animation:
  - Use `AnimationController` with 4s duration, repeat
  - Lissajous curve: `x = sin(t)`, `y = sin(2t)` scaled to ~80dp amplitude
  - Render a device silhouette icon (`Icons.phone_android`) translated along the path
  - Wrap in `AnimatedBuilder`
- [ ] Confidence ring:
  - `CustomPaint` drawing a circular arc from 0 to `confidence * 2 * pi`
  - Ring radius: ~60dp, stroke width: 8dp
  - Color: `confidence < 0.4 ? hudDanger : confidence < 0.8 ? hudWarning : hudPrimary`
  - Center text: `"${(confidence * 100).toInt()}%"` in `hudFontFamily`, 24pt
- [ ] Instruction text: below the ring, `hudPrimary` color, 14pt
- [ ] "Done" button: `ElevatedButton` with `fabBackground` fill; `onPressed` is null when confidence < 0.8
- [ ] "Restart" button: `TextButton` with `hudSecondary` color; calls `onRestart` callback
- [ ] Accept `confidenceProvider` or `double confidence` + `VoidCallback onDone` + `VoidCallback onRestart`
- [ ] For standalone testing: include a mock mode with a `Timer.periodic` that increments confidence from 0 to 1 over 10 seconds
- [ ] Widget test: render with confidence=0.5, verify Done disabled and ring is hudWarning; render with 0.9, verify Done enabled and ring is hudPrimary
