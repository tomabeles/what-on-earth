# WOE-071: Telemetry HUD -- Heading Tape

**ID:** WOE-071
**Phase:** UI -- Telemetry HUD
**Depends on:** WOE-070
**UI Spec:** SS5.2

---

## Description

Add the horizontal scrolling heading tape to the top-center of the HUD. Displays 60 degrees of arc centered on the current heading with major ticks every 10 degrees and minor ticks every 5 degrees. Cardinal/intercardinal letters at standard bearings.

Uses mock heading data (static or slowly rotating) until WOE-018 (DeviceOrientation) and WOE-019 (orientation wiring) are completed.

---

## Definition of Done

- [ ] Heading tape renders at top-center of the HUD CustomPaint
- [ ] Shows 60 degrees of visible arc centered on current heading
- [ ] Major tick every 10 degrees with numeric label
- [ ] Minor tick every 5 degrees (no label)
- [ ] Cardinal letters: N (0), NE (45), E (90), SE (135), S (180), SW (225), W (270), NW (315)
- [ ] Current heading shown in a downward-pointing notch above tape center
- [ ] Background: `hudBackground` strip behind the tape
- [ ] Color: `hudPrimary` for ticks and labels; `hudFontFamily` for text
- [ ] Accepts `double headingDeg` parameter (0-360)
- [ ] Test: verify tick count and cardinal labels render correctly for heading=0

---

## Sub-tasks

- [ ] Add `paintHeadingTape(Canvas, Size, double headingDeg)` method to `HudPainter`
- [ ] Calculate visible range: `[heading - 30, heading + 30]` with wrapping at 0/360
- [ ] Draw background strip: `canvas.drawRect` with `hudBackground` color, height ~40dp, top margin ~20dp from safe area
- [ ] Iterate degrees in visible range; for each degree divisible by 5, draw a minor tick (6dp tall); for each divisible by 10, draw a major tick (12dp tall) + label
- [ ] Map degree position to X coordinate: `(degree - (heading - 30)) / 60 * screenWidth`
- [ ] Draw cardinal letters at their degree positions (replace numeric label)
- [ ] Draw center notch: small downward triangle at horizontal center, above the tape
- [ ] Draw current heading value text inside the notch (e.g., "247")
- [ ] Add `headingDeg` field to `HudPainter` constructor; pass from `TelemetryHud` widget
- [ ] For now: use a static mock heading (e.g., 247.0) or a `Timer` that slowly rotates for visual testing
- [ ] Test: instantiate `HudPainter(headingDeg: 0)`, verify N label is centered
