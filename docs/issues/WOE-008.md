# WOE-008: OrbitalPosition Model and PositionSource Interface

**ID:** WOE-008
**Phase:** 2 — Position Module
**Step:** 2.1
**Depends on:** WOE-001

---

## Description

Define the core data model and abstract interface for the position module. The `OrbitalPosition` class and `PositionSource` abstract class are the contract that all three concrete position sources (`ISSLiveSource`, `TLESource`, `StaticSource`) implement. Getting this interface right before implementing the sources is important — it cannot change without updating all implementations.

**Reference:** TECH_SPEC §7.1
