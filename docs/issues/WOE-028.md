# WOE-028: shelf Tile Server in Background Isolate

**ID:** WOE-028
**Phase:** 4 — Map Layers
**Step:** 4.4
**Depends on:** WOE-001

---

## Description

Start the `shelf` HTTP server on `localhost:8765` at app launch inside a dedicated background Dart `Isolate`. This server is the delivery mechanism for pre-downloaded raster map tiles stored in the app's documents directory. It must run for the entire app lifecycle and handle tile requests with appropriate 200/404 responses.

**Reference:** TECH_SPEC §4.5, §7.3

---

## Definition of Done

- [ ] `TileServer` class exists in `lib/tile_cache/tile_server.dart`
- [ ] On `start()`, spawns a background isolate running a `shelf` server on port 8765
- [ ] Route `GET /tiles/{layer}/{z}/{x}/{y}.webp` serves the file from `{documents}/tiles/{layer}/{z}/{x}/{y}.webp` or returns 404
- [ ] Route `GET /tiles/{layer}/{z}/{x}/{y}.png` serves the `.png` equivalent
- [ ] Server starts before `GlobeView` initializes (app startup order enforced)
- [ ] Server stops cleanly on app termination
- [ ] A `TileServerStatus` (running, port) is exposed for the settings screen
- [ ] Manual test: `curl http://localhost:8765/tiles/base/3/4/2.webp` from the device returns 200 or 404 (not an error)

---

## Sub-tasks

- [ ] Create `lib/tile_cache/tile_server.dart`
- [ ] Define the isolate entry point function that:
  - Gets the app documents directory path (passed via `SendPort` from main isolate since `path_provider` is not available in isolates directly)
  - Creates a `shelf` pipeline: `shelf_router.Router()` with routes for `/tiles/<layer>/<z>/<x>/<y>.<ext>`
  - Handler: construct `File(path)` from `{documents}/tiles/{layer}/{z}/{x}/{y}.{ext}`; return `shelf.Response.ok(file.openRead(), headers: {'Content-Type': mimeType})` or `shelf.Response.notFound('tile not found')`
  - Listen with `shelf_io.serve(handler, 'localhost', 8765)`
- [ ] In `main.dart`, call `TileServer.start(documentsPath)` before `runApp`
- [ ] Pass `documentsPath` to the isolate: use `path_provider.getApplicationDocumentsDirectory()` in main, pass result path string to isolate via `Isolate.spawn` init message
- [ ] Add `TILE_SERVER_PORT` dart-define default 8765
- [ ] Manual test instructions documented in `web_globe/README.md`
