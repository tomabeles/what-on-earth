# WOE-002: CesiumJS Vite Build Pipeline

**ID:** WOE-002
**Phase:** 1 — Foundation
**Step:** 1.2
**Depends on:** WOE-001

---

## Description

Create the `web_globe/` Vite project that produces the CesiumJS bundle loaded by the Flutter WebView. This pipeline must bundle CesiumJS 1.138.0 and `satellite.js` offline — no CDN — and output all static files (including the mandatory `Workers/`, `Assets/`, `ThirdParty/`, `Widgets/` directories) into `assets/globe/`.

The built output is git-ignored and regenerated by CI. A `scripts/build_globe.sh` shell script wraps the build so CI and local developers invoke it the same way.

**Reference:** TECH_SPEC §3.2, §6.3

---

## Definition of Done

- [ ] `web_globe/` directory exists with a working Vite project
- [ ] `npm ci && npx vite build` completes without errors
- [ ] Build output in `assets/globe/` contains: `index.html`, bundled JS, and all four Cesium static directories (`Workers/`, `Assets/`, `ThirdParty/`, `Widgets/`)
- [ ] `window.CESIUM_BASE_URL` is set to `/` in the bundle
- [ ] `satellite.js` is importable from the bundle (verify with a `console.log(typeof satellite)` in `main.js`)
- [ ] `assets/globe/` is listed in `.gitignore`; `assets/globe/.gitkeep` is committed
- [ ] `scripts/build_globe.sh` exists, is executable, and runs the build end-to-end
- [ ] A `web_globe/README.md` documents how to run the build locally

---

## Sub-tasks

- [ ] Create `web_globe/` directory and run `npm init -y`
- [ ] Install dependencies: `npm install cesium@1.138.0 satellite.js@4.1.4`
- [ ] Install dev dependencies: `npm install -D vite@^6.1.0 vite-plugin-cesium@^1.3.3`
- [ ] Create `web_globe/vite.config.js` using `vite-plugin-cesium`; set `outDir: '../assets/globe'` and `base: '/'`
- [ ] Create `web_globe/index.html` with a `<div id="cesiumContainer">` and a `<script type="module" src="/src/main.js">`
- [ ] Create `web_globe/src/main.js` that sets `window.CESIUM_BASE_URL = '/'` before importing CesiumJS; imports `satellite.js`; logs `'globe_bundle_loaded'` to console
- [ ] Run `npx vite build` and verify `assets/globe/` output structure
- [ ] Add `assets/globe/` to `.gitignore` (but not `assets/globe/.gitkeep`)
- [ ] Create `scripts/build_globe.sh`: `cd web_globe && npm ci && npx vite build --outDir ../assets/globe`
- [ ] Run `chmod +x scripts/build_globe.sh`
- [ ] Verify the bundle size is reasonable (CesiumJS full build is ~10–15 MB minified JS + workers)
