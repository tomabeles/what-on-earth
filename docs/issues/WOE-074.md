# WOE-074: Telemetry HUD -- Data Strip

**ID:** WOE-074
**Phase:** UI -- Telemetry HUD
**Depends on:** WOE-070
**UI Spec:** SS5.2

---

## Description

Add the data strip readout panels to the bottom of the HUD. Two columns of labeled values showing ISS position, device orientation, velocity, source, and data age. Positioned above the Controls button and NAV FAB.

Uses mock data until position controller (WOE-013) and sensor fusion (WOE-018) are wired.

---

## Definition of Done

- [ ] Data strip renders at the bottom of the HUD, above the chrome layer
- [ ] Left column: LAT, LON, ALT, HDG
- [ ] Right column: VEL, SRC, AGE, PCH, ROL (or similar split for balance)
- [ ] Each column has a `hudBackground` rounded rect backing
- [ ] Label color: `hudSecondary`; value color: `hudPrimary`
- [ ] Font: `hudFontFamily` at `hudFontSize`
- [ ] Value formatting: LAT as `12.345 N`, LON as `98.762 W`, ALT as `420.1 km`, HDG as `247`, PCH as `+12`, ROL as `+3`, VEL as `7.66 km/s`, SRC as colored dot + label, AGE as `2s`
- [ ] VEL blank when source is TLE or Static
- [ ] AGE hidden when source is Static
- [ ] Accepts data model with all fields (nullable for missing data)
- [ ] Test: verify correct formatting for sample values

---

## Sub-tasks

- [ ] Add `paintDataStrip(Canvas, Size, HudData data)` to `HudPainter`
- [ ] Define `HudData` class: `latDeg`, `lonDeg`, `altKm`, `headingDeg`, `pitchDeg`, `rollDeg`, `velocityKmS`, `sourceType`, `ageSeconds` (all nullable doubles/ints)
- [ ] Left column background: `canvas.drawRRect` with `hudBackground` color, bottom-left area
- [ ] Right column background: same, bottom-right area
- [ ] Layout: fixed Y positions for each row; use `TextPainter` for each label + value pair
- [ ] Label rendering: `TextPainter` with `TextStyle(fontFamily: hudFontFamily, fontSize: hudFontSize - 2, color: hudSecondary)`
- [ ] Value rendering: `TextPainter` with `TextStyle(fontFamily: hudFontFamily, fontSize: hudFontSize, color: hudPrimary)`
- [ ] Format helpers: `_formatLat(double)` -> `"12.345 N"`, `_formatLon(double)` -> `"98.762 W"`, etc.
- [ ] SRC row: draw colored dot (8dp) + text; dot color from `statusLive/statusEstimated/statusOffline`
- [ ] AGE row: show "Xs" or "--" if null; hide entirely for static source
- [ ] VEL row: show formatted value or "--" for TLE/static
- [ ] Add `HudData` to `HudPainter` constructor; populate from providers or mock
- [ ] Test: format helpers return correct strings
