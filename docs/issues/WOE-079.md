# WOE-079: 2D Map -- ISS Ground Track + Viewing Footprint

**ID:** WOE-079
**Phase:** UI -- Screens
**Depends on:** WOE-078, WOE-011
**UI Spec:** SS4.2

---

## Description

Add the ISS ground track polyline and viewing footprint ellipse to the 2D Map View. The ground track shows the next 90-minute orbital path as a CesiumJS `Polyline`. The footprint shows the current ground-level viewing circle as a semi-transparent `Ellipse` entity (~2,200 km radius at 400 km altitude, 10 degree min elevation).

Both are computed in JavaScript using `satellite.js` from the current TLE data.

---

## Definition of Done

- [ ] Ground track: `Polyline` entity showing ~90 minutes of future orbital path
- [ ] Polyline updates when position updates (re-computed periodically, not every frame)
- [ ] Footprint: semi-transparent `Ellipse` entity centered on current ISS position
- [ ] Footprint radius: ~2,200 km (adjusts slightly with altitude)
- [ ] Both entities visible only in `map` mode; hidden in `ar` mode
- [ ] Ground track color: `hudPrimary` (passed from Flutter or hardcoded for V1)
- [ ] Footprint color: `hudPrimary` at 15% opacity fill, 50% opacity outline

---

## Sub-tasks

- [ ] In `web_globe/src/main.js` (or a new `ground_track.js` module):
  - `computeGroundTrack(satrec, minutesAhead=90, stepMinutes=1)`: use `satellite.js` to propagate from now to now+90min in 1-min steps; return array of `{lat, lon}`
  - `updateGroundTrack()`: remove old entity, create new `Polyline` entity from computed points
  - `updateFootprint(lat, lon, altKm)`: compute footprint radius from altitude and min elevation angle (10 deg); create/update `Ellipse` entity
- [ ] Footprint radius formula: `R_earth * arccos(R_earth / (R_earth + altKm) * cos(minElev)) - R_earth * minElev` (simplified to ~2,200 km at 400 km)
- [ ] Call `updateGroundTrack()` on every `UPDATE_POSITION` message (or throttle to every 30s)
- [ ] Call `updateFootprint()` on every `UPDATE_POSITION` message
- [ ] In `SET_MODE` handler: show entities when mode='map', hide when mode='ar'
- [ ] Run `scripts/build_globe.sh`
- [ ] Manual test: open 2D map, verify polyline traces orbital path and ellipse is visible
