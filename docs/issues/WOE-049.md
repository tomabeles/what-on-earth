## Description

Implement the Settings screen that consolidates all user-configurable options: position source selection, layer toggles, tile cache status, account management, data attribution, and about information.

**Reference:** TECH_SPEC Â§10.2

---

## Definition of Done

- [ ] `SettingsScreen` exists in `lib/screens/settings_screen.dart`
- [ ] Navigable from the AR view via a settings icon button
- [ ] **Position Source section:** radio buttons for Live ISS, Static (with lat/lon/alt input fields, enabled when Static is selected), TLE/Estimated only
- [ ] **Layers section:** toggle switches for each layer (mirrors the layer toggle panel in AR view; in sync via shared state)
- [ ] **Tile Cache section:** shows current cache size, last sync time, "Re-download" button, "Clear Cache" button (with confirmation dialog)
- [ ] **Account section:** shows signed-in email + "Sign Out", or "Sign In" button if not authenticated
- [ ] **Calibration section:** "Recalibrate compass" button that navigates to `CalibrationScreen`
- [ ] **Data Attribution section:** lists all data sources with license names (fulfills PRD open data registry requirement)
- [ ] **About section:** app version, links to privacy policy
- [ ] All settings persist across app restarts

---

## Sub-tasks

- [ ] Create `lib/screens/settings_screen.dart` using `Scaffold` with `ListView` of `ListTile` groups separated by `Divider`
- [ ] Position source: `ListTile` with a trailing `DropdownButton<PositionSourceType>`; when `static` selected, expand to show lat/lon/alt `TextFormField`s; on change, call `ref.read(positionControllerProvider.notifier).setSourceMode(...)`
- [ ] Layers: use the same shared preferences state as the layer toggle panel; changes propagate to the globe immediately
- [ ] Tile cache: read from `TileCacheManager.getCacheStatus()`; format `totalSizeBytes` as "X.X GB"; format `lastSyncAt` as relative time; "Re-download" re-triggers `TileDownloadStep`; "Clear Cache" calls `TileCacheManager.clearAll()` with confirmation `AlertDialog`
- [ ] Account: watch `authStateProvider`; display user email or "Not signed in"; "Sign In" navigates to `SignInScreen`; "Sign Out" calls `supabase.auth.signOut()`
- [ ] Calibration: `ListTile` with title "Compass calibration" and subtitle showing last calibration date from `CalibrationStore.load()?.calibratedAt`; trailing "Recalibrate" button
- [ ] Data attribution: static `ExpansionTile` listing: OpenStreetMap (ODbL), Natural Earth (public domain), CelesTrak (public domain), wheretheiss.at (attribution required)
- [ ] About: `ListTile` showing `PackageInfo.fromPlatform().version`
- [ ] Add a settings `IconButton` to the AR view's UI chrome layer
