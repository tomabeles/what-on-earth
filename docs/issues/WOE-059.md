# WOE-059: App Store Submission Preparation

**ID:** WOE-059
**Phase:** 7 — NFR Pass and Polish
**Step:** 7.10
**Depends on:** WOE-056, WOE-058

---

## Description

Prepare all assets, metadata, and compliance materials required for Apple App Store and Google Play Store submission. This includes privacy policy, screenshots, app review notes explaining the ISS use case and sensor permission justifications, and age rating questionnaire.

**Reference:** TECH_SPEC §11.1, §11.3

---

## Definition of Done

- [ ] Apple App Store Connect listing created with all required metadata
- [ ] Google Play Console listing created with all required metadata
- [ ] Privacy policy published at a stable URL and linked in both store listings
- [ ] App screenshots (at least 3 per required device size) prepared for both stores
- [ ] App review notes written explaining: ISS use case, why camera is needed (AR), why motion sensors are needed (device orientation), why location is NOT accessed
- [ ] App privacy questionnaire completed (no location collection, no third-party analytics by default, optional crash reporting disclosed)
- [ ] Age rating set to 4+ / Everyone
- [ ] A `docs/store_submission.md` documents the listing metadata for future updates

---

## Sub-tasks

- [ ] Write a privacy policy covering: what data is collected (pins stored locally; optionally backed up to Supabase if user signs in; no location data collected; optional crash reports if opt-in), how data is used, data retention, contact info. Publish at `https://whatonearth.app/privacy` (or equivalent URL)
- [ ] Prepare screenshots:
  - AR view with globe overlaid on a viewport view
  - 2D map view showing the ISS ground track
  - Pin detail screen showing next pass info
  - Onboarding step showing offline download
  - Screenshots for iPhone 6.9" (required), iPhone 6.5", iPad Pro 13" (required), Android phone and tablet
- [ ] App Store Connect:
  - App name: "What On Earth?!"
  - Subtitle: "AR Earth Viewer for Orbit"
  - Description: 2–3 paragraphs from PRFAQ press release, adapted for store format
  - Keywords: astronaut, ISS, earth, AR, augmented reality, space, globe, orbit
  - Category: Navigation or Utilities
  - Review notes: "This app is designed for ISS crew members. The camera permission is required for the AR overlay. Motion sensors are required to determine device orientation. No GPS or location data is used."
- [ ] Google Play Console:
  - Same metadata as App Store
  - Content rating questionnaire: no violence, no sexual content, no location sharing → Everyone
  - App permissions declaration: camera (AR), motion sensors (orientation)
- [ ] Submit for review; monitor review status
- [ ] Commit `docs/store_submission.md` with all listing metadata for future reference
