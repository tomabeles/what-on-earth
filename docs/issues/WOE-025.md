# WOE-025: Natural Earth GeoJSON Download and Preprocessing

**ID:** WOE-025
**Phase:** 4 — Map Layers
**Step:** 4.1
**Depends on:** WOE-001

---

## Description

Download the Natural Earth GeoJSON source files for country borders, coastlines, lakes, and populated places. Run the simplification script to reduce file size (removing precision beyond what is visible at orbital zoom levels), and commit the processed files to `assets/geodata/`.

These files are the primary source for all vector annotation layers on the globe.

**Reference:** TECH_SPEC §3.5, §6.1

---

## Definition of Done

- [ ] The following files exist in `assets/geodata/` and are committed:
  - `ne_10m_admin_0_countries.geojson` — country polygons/borders
  - `ne_10m_coastline.geojson` — coastline linestrings
  - `ne_10m_lakes.geojson` — lake polygons
  - `ne_10m_populated_places.geojson` — city/settlement points with `NAME` and `POP_MAX` properties
- [ ] All files have been simplified to a coordinate precision of 2 decimal places (≈ 1.1 km — more than sufficient at orbital zoom)
- [ ] Combined file size of all four GeoJSON files is ≤ 25 MB
- [ ] `scripts/preprocess_geodata.sh` is committed and documents the simplification commands
- [ ] `flutter pub get` and `flutter build apk --debug` still succeed with the new assets

---

## Sub-tasks

- [ ] Download source files from `https://naturalearth.com/downloads` (10m resolution):
  - `ne_10m_admin_0_countries`
  - `ne_10m_coastline`
  - `ne_10m_lakes`
  - `ne_10m_populated_places`
  - (Natural Earth provides as shapefiles; convert to GeoJSON using `ogr2ogr` or download the GeoJSON format directly)
- [ ] Install `mapshaper` CLI: `npm install -g mapshaper`
- [ ] Create `scripts/preprocess_geodata.sh`:
  ```bash
  mapshaper ne_10m_admin_0_countries.geojson -simplify 10% -o precision=0.01 format=geojson assets/geodata/ne_10m_admin_0_countries.geojson
  # repeat for other files with appropriate simplification %
  ```
- [ ] For `ne_10m_populated_places`: filter to `POP_MAX > 100000` to reduce file size while keeping major cities
- [ ] Remove unnecessary properties from each file (keep only `NAME`, `SOV_A3` for countries; `NAME`, `POP_MAX` for cities)
- [ ] Verify file sizes: total ≤ 25 MB; individual files should be well under 10 MB each
- [ ] Commit processed files to `assets/geodata/`
