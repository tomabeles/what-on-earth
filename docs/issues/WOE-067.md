# WOE-067: Status Bar Widget

**ID:** WOE-067
**Phase:** UI -- AR Chrome
**Depends on:** WOE-063
**Supersedes:** WOE-015
**UI Spec:** SS5.1

---

## Description

Implement the compact semi-transparent pill bar pinned to the top of the AR View and 2D Map View. Always visible, not interactive in V1. Displays: position source dot + label, data age, connectivity icon, and tile cache freshness warning.

Can use mock/static data until WOE-013 (PositionController) is wired.

---

## Definition of Done

- [ ] `StatusBar` widget in `lib/shared/status_bar.dart`
- [ ] Height 28dp, horizontal padding 12dp, corner radius 14dp (full pill)
- [ ] Background: `surfaceOverlay`; top margin 12dp from safe area
- [ ] Source dot: filled circle colored `statusLive` / `statusEstimated` / `statusOffline`
- [ ] Source label: "ISS Live" / "TLE Estimated" / "Static"
- [ ] Age: "Xs ago" -- hidden when source is Static or age < 1s; updates every second
- [ ] Connectivity icon: wifi/cell icon when online; warning icon in `statusOffline` when offline
- [ ] Tile freshness: hidden unless tile cache > 30 days stale; shows map icon + "Stale" in `hudWarning`
- [ ] Typography: `hudFontFamily`, `hudFontSize - 1` pt, color `hudPrimary`
- [ ] Widget test: correct label and dot color for each source type

---

## Sub-tasks

- [ ] Create/replace `lib/shared/status_bar.dart`
- [ ] Accept `PositionSourceStatus` as parameter (or watch provider if available; fall back to static mock)
- [ ] Layout: `Container` with `BoxDecoration(color: tokens.surfaceOverlay, borderRadius: 14)`, child is a `Row` of elements separated by `Text(' . ')` or `SizedBox`
- [ ] Source dot: `Container` 8dp circle with appropriate status color
- [ ] Source label: `Text` with `hudFontFamily`
- [ ] Age: `StreamBuilder` or `Timer.periodic` updating "Xs ago" text every second
- [ ] Connectivity: use `connectivity_plus` stream; show `Icons.wifi` or `Icons.wifi_off`
- [ ] Tile freshness: accept optional `DateTime? lastTileSync`; show warning if > 30 days or null
- [ ] Position inside AR view: `Positioned(top: MediaQuery.of(context).padding.top + 12, left: 12, right: 12)`
- [ ] Widget test: render with mock `PositionSourceStatus(type: live)`, verify green dot and "ISS Live"
