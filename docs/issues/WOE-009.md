# WOE-009: ISSLiveSource Implementation

**ID:** WOE-009
**Phase:** 2 — Position Module
**Step:** 2.2
**Depends on:** WOE-008

---

## Description

Implement the `ISSLiveSource` concrete `PositionSource` that polls `https://api.wheretheiss.at/v1/satellites/25544` every 2 seconds using `dio`. The source must handle all error conditions gracefully: non-2xx responses, timeouts, and network errors all result in the last known position being re-emitted tagged as `estimated` (rather than throwing or going silent).

**Reference:** TECH_SPEC §4.1, §7.1

---

## Definition of Done

- [ ] `ISSLiveSource` implements `PositionSource` interface
- [ ] Successful response emits `OrbitalPosition` with `sourceType: live`
- [ ] Any error (network, timeout, non-2xx) emits the last known position with `sourceType: estimated`; if no last known position exists yet, the stream stays silent until first success
- [ ] `dio` request timeout is set to 5 seconds
- [ ] `stop()` cancels the polling timer and closes the stream
- [ ] Unit tests with a mocked `dio` client covering: successful response, network error, timeout, 500 response
- [ ] `flutter analyze` passes on the file

---

## Sub-tasks

- [ ] Create `lib/position/iss_live_source.dart`
- [ ] Inject a `Dio` instance (not constructed internally — injectable for testing)
- [ ] Use a `StreamController<OrbitalPosition>` with `broadcast: true`
- [ ] On `start()`, create a `Timer.periodic(Duration(seconds: 2), ...)` that calls a private `_poll()` method
- [ ] `_poll()`: call `https://api.wheretheiss.at/v1/satellites/25544`, parse response JSON (`latitude`, `longitude`, `altitude` in km), emit `OrbitalPosition` with `sourceType: live`. On error, emit last known position with `sourceType: estimated` (or skip if none yet)
- [ ] Set `dio.options.connectTimeout = Duration(seconds: 5)` and `receiveTimeout = Duration(seconds: 5)`
- [ ] On `stop()`: cancel timer, close stream controller
- [ ] Write `test/unit/iss_live_source_test.dart` with mocked `Dio` (use `mockito` or `http_mock_adapter`)
  - Test: successful poll emits live position
  - Test: network error emits last known as estimated
  - Test: no last known position → stream is silent on first error
  - Test: stop() cancels polling
