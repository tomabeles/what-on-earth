# WOE-063: Theme System -- AppTokens + 4 Named Themes

**ID:** WOE-063
**Phase:** UI -- Theme Foundation
**Depends on:** --
**UI Spec:** SS2

---

## Description

Implement the full theme system as defined in UI_SPEC SS2. This includes the `AppTokens` theme extension, the `AppTheme` wrapper class, `AppThemeRegistry` for theme discovery, and all four named themes (night, dark, starwars, startrek) with exact color values from SS2.3. Also add the JetBrainsMono font for HUD numerics.

Wire the default theme (`night`) into `app.dart` so that `Theme.of(context).extension<AppTokens>()` is available throughout the app.

---

## Definition of Done

- [ ] `AppTokens extends ThemeExtension<AppTokens>` with all 16 tokens from SS2.2
- [ ] `copyWith()` and `lerp()` implemented for theme animation support
- [ ] `AppTheme` class with `id`, `displayName`, `tokens` fields
- [ ] `AppThemeRegistry.themes` lists all 4 themes; `find(id)` returns theme or default
- [ ] Named themes defined with exact hex values from SS2.3: `night`, `dark`, `starwars`, `startrek`
- [ ] JetBrainsMono font available (via `google_fonts` or bundled in `assets/fonts/`)
- [ ] `app.dart` applies the `night` theme by default via `ThemeData.extension<AppTokens>()`
- [ ] `flutter analyze` clean
- [ ] Unit tests for `AppThemeRegistry.find()` and `AppTokens.lerp()`
