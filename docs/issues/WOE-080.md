# WOE-080: Settings -- Display Section

**ID:** WOE-080
**Phase:** UI -- Settings
**Depends on:** WOE-064
**UI Spec:** SS4.6

---

## Description

Implement the Settings screen shell and the Display section. The Display section contains: a horizontal scrollable theme picker showing all registered themes with name and accent color swatch, and a Telemetry HUD toggle.

This is the first settings section. The screen shell (`Scaffold`, `ListView`, grouped cards) is created here and reused by subsequent settings tickets.

---

## Definition of Done

- [ ] `SettingsScreen` in `lib/screens/settings_screen.dart` (replace placeholder)
- [ ] Scaffold with `surfacePrimary` background, AppBar with "Settings" title
- [ ] Grouped list using `surfaceSecondary` card style per section
- [ ] **Display section** with:
  - Theme picker: horizontal scrollable row of cards, one per `AppThemeRegistry.themes`
  - Each card shows theme `displayName` + accent color swatch (`hudPrimary`)
  - Selected theme has a border or check mark
  - Tapping a theme calls `themeProvider.setTheme(id)`
  - Telemetry HUD toggle: `Switch` wired to `hudVisibilityProvider`
- [ ] NAV FAB visible with Settings highlighted
- [ ] Widget test: theme picker shows all 4 themes; tapping a theme updates provider
