# WOE-036: MAP_TAP Bridge Message and Add Pin Bottom Sheet Trigger

**ID:** WOE-036
**Phase:** 5 — Pins
**Step:** 5.3
**Depends on:** WOE-006, WOE-035

---

## Description

Implement the `MAP_TAP` JS→Flutter bridge message so that when a user taps the CesiumJS globe, the geographic coordinates of the tap are sent to Flutter and the "Add Pin" bottom sheet is triggered.

**Reference:** TECH_SPEC §8.2

---

## Definition of Done

- [ ] CesiumJS detects globe taps using `viewer.screenSpaceEventHandler`
- [ ] On tap, picks the globe surface at the screen position and computes geographic lat/lon
- [ ] `MAP_TAP` message sent to Flutter with `{ lat, lon }`
- [ ] Flutter receives the message and shows the `AddPinSheet` bottom sheet (stub acceptable — WOE-037 implements the full UI)
- [ ] Taps on sky (no globe surface) are ignored gracefully (no message sent)
- [ ] Works in both AR view and 2D map view

---

## Sub-tasks

- [ ] In `web_globe/src/main.js`, set up a `Cesium.ScreenSpaceEventHandler` on `viewer.canvas`:
  ```javascript
  const handler = new Cesium.ScreenSpaceEventHandler(viewer.canvas);
  handler.setInputAction((event) => {
    const cartesian = viewer.camera.pickEllipsoid(event.position, viewer.scene.globe.ellipsoid);
    if (!cartesian) return;  // tapped sky
    const cartographic = Cesium.Cartographic.fromCartesian(cartesian);
    const lat = Cesium.Math.toDegrees(cartographic.latitude);
    const lon = Cesium.Math.toDegrees(cartographic.longitude);
    window.flutter_inappwebview.callHandler('MAP_TAP', { lat, lon });
  }, Cesium.ScreenSpaceEventType.LEFT_CLICK);
  ```
- [ ] Register `MAP_TAP` handler in Flutter `BridgeController.registerHandlers()`: parse `{lat, lon}`, call a callback provided to the bridge
- [ ] In `ARView`, provide a `onMapTap(double lat, double lon)` callback to `GlobeView` that calls `showModalBottomSheet` with an `AddPinSheet` stub
- [ ] `AddPinSheet` stub: displays `'Tapped: ${lat.toStringAsFixed(4)}, ${lon.toStringAsFixed(4)}'` with a close button
- [ ] Test on device: tap the globe, verify bottom sheet appears with correct coordinates
