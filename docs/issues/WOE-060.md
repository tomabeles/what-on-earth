# WOE-060: StaticPositionSource

**ID:** WOE-060
**Phase:** 2 — Position Module
**Step:** 2.0 (prerequisite)
**Depends on:** WOE-008

---

## Description

Implement `StaticPositionSource`, a concrete `PositionSource` that repeatedly emits a single fixed `OrbitalPosition` at a configurable interval. It requires no network access, making it the ideal stub for simulator testing and for exercising the full Flutter → bridge → CesiumJS data path before the live or TLE sources exist.

The static position defaults to a visually obvious location (ISS orbital altitude over London: 51.5°N, −0.1°E, 420 km) so that a first-time device run immediately places a satellite marker on the globe without any configuration.

**Reference:** TECH_SPEC §7.1

---

## Definition of Done

- [ ] `lib/position/static_position_source.dart` compiles without errors and `flutter analyze` is clean
- [ ] Implements `PositionSource` interface from WOE-008
- [ ] `type` getter returns `PositionSourceType.static`
- [ ] `start()` begins emitting the configured `OrbitalPosition` on a timer at the given interval; default interval is 5 seconds
- [ ] Every emission uses `sourceType: PositionSourceType.static`
- [ ] `stop()` cancels the timer and closes the stream controller
- [ ] Constructor accepts optional `position` and `interval` parameters for test overrides
- [ ] Unit tests cover: emission on tick, correct `sourceType`, `stop()` silences the stream

---

## Sub-tasks

- [ ] Create `lib/position/static_position_source.dart`:
  - Constructor: `StaticPositionSource({ OrbitalPosition? position, Duration interval = const Duration(seconds: 5) })`
  - Default position: `OrbitalPosition(latDeg: 51.5, lonDeg: -0.1, altKm: 420.0, timestamp: DateTime.now(), sourceType: PositionSourceType.static)`
  - `StreamController<OrbitalPosition>` with `broadcast: true`
  - `start()`: creates `Timer.periodic(interval, (_) => _controller.add(_position))`, also emits one event immediately before starting the timer so the globe updates without waiting for the first tick
  - `stop()`: cancels timer, closes stream controller
- [ ] Write `test/unit/static_position_source_test.dart`:
  - Verify first emission arrives immediately after `start()`
  - Verify `sourceType` is always `PositionSourceType.static`
  - Verify stream goes silent after `stop()`
  - Verify custom `position` and `interval` are respected
